# My i3wm Dotfiles

My personal i3wm setup dotfiles running on Gentoo Linux.

## Info
- **Creator:** minzo181
- **OS:** Gentoo Linux
- **WM:** i3wm
- **Terminal:** Alacritty
- **Bar:** Polybar
- **Launcher:** Rofi

## Structure
- `i3/` — main i3 config, keybindings, autostart
- `polybar/` — bar config + launch script
- `alacritty/` — terminal config
- `rofi/` — launcher theme and config

## Requirements
- i3wm
- Polybar
- Alacritty
- Rofi
- Nerd Fonts (for Polybar icons)
- feh (for wallpaper)

## Installation
Copy the folders to `~/.config/`:
\```bash
cp -r i3 ~/.config/
cp -r polybar ~/.config/
cp -r alacritty ~/.config/
cp -r rofi ~/.config/
\```
